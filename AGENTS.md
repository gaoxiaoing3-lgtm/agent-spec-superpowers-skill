# Agent Instructions

## Required Order

Follow this order exactly:

1. Spec Kit: read constitution and feature specs; update `spec.md`, `plan.md`, and `tasks.md` before meaningful implementation.
2. Superpowers: clarify, plan, implement in small vertical slices, verify, review, and hand off.
3. Agent Skill: use the relevant `skills/<name>/SKILL.md` only after the spec and execution discipline are established.

## Read Order

1. `.specify/memory/constitution.md`
2. `specs/001-integrated-agent-workflow/spec.md`
3. `specs/001-integrated-agent-workflow/plan.md`
4. `specs/001-integrated-agent-workflow/tasks.md`
5. `skills/agent-spec-superpowers/SKILL.md`

## Hard Rule

A skill is never the first source of truth for feature work. If the spec is missing or stale, fix the spec first.
